# Ride Data Analysis Dashboard (OLA Dataset)

## 📊 Project Overview
This project focuses on analyzing ride-booking data to understand customer behavior, trip patterns, and demand trends using Power BI.

## 🛠 Tools Used
- Power BI
- MS Excel
- Power Query

## 🔍 Key Features
- Data cleaning and transformation
- Dashboard creation and visualization
- Peak hour analysis
- Customer demand trends
