# Amazon Sales Analysis Dashboard

## 📊 Project Overview
This project focuses on analyzing e-commerce sales data to identify revenue trends, product performance, and customer purchasing patterns using Power BI.

## 🛠 Tools Used
- Power BI  
- DAX  
- MS Excel  

## 🔍 Key Features
- Performed data analysis on sales data across regions and categories  
- Created interactive dashboards for data visualization  
- Used DAX functions to calculate key performance indicators (KPIs)  
- Analyzed seasonal trends and top-performing products  

## 📈 KPIs Analyzed
- Total Revenue  
- Profit  
- Average Order Value  
- Top-Selling Products
